## 2. 왜 필요한가?

### 2-1. 신안군 현황

- **인구**: 약 35,000명 (65세 이상 40% 이상)
- **섬**: 1,028개 (유인도 72개)
- **주요 산업**: 천일염, 수산물(전복·김·새우젓), 농산물(대파·시금치·양파)
- **관광**: 퍼플섬, 1004섬 브랜드 (연간 관광객 200만명+)

### 2-2. 농어민이 겪는 문제

| 문제 | 현실 |
|------|------|
| 판로 부족 | 중간상인에게 헐값에 넘기거나, 판매 자체를 포기 |
| 디지털 소외 | 기존 온라인 쇼핑몰은 상품 등록이 너무 복잡 (사진 촬영, 상세 설명 작성, 카테고리 설정 등) |
| 브랜드 부재 | 개별 농어민은 브랜드 파워가 없어 가격 경쟁력 확보 어려움 |
| 물류 한계 | 섬 지역 특성상 배송 비용이 높고 시간이 오래 걸림 |

### 2-3. 기존 시도의 한계

| 기존 방식 | 문제점 |
|-----------|--------|
| 지자체 쇼핑몰 | 상품 등록이 어려워 실질 이용률 10% 미만 |
| 쿠팡/네이버 | 수수료 10-15%, 소통 불가, 대기업 위주 |
| 직거래 장터 | 비정기적, 물리적 이동 필요, 규모 한계 |
| SNS 판매 | 체계 없음, 결제·배송 관리 어려움 |

## 3. 고향마켓이란?

### 3-1. 핵심 콘셉트

> **"사진 한 장이면 됩니다"**
> AI가 사진을 분석하여 상품명, 설명, 가격, 카테고리를 자동 생성합니다.
> 디지털에 익숙하지 않은 어르신도 **3분 안에 상품 등록이 가능**합니다.

### 3-2. 4대 핵심 가치

#### 1) AI 자동 상품 등록 (디지털 문턱 제로)
```
할머니가 고추장 사진을 찍음
    ↓
AI가 자동으로 분석
    ↓
상품명: "신안 전통 고추장"
설명: "갯바람 맞으며 3년 숙성한 전통 고추장..."
가격: 15,000원 / 1kg
    ↓
[등록] 버튼만 누르면 끝!
```

#### 2) 감성 기반 AI 추천 (고향이 그리울 때)
- 기존 쇼핑몰: "이 상품을 본 사람이 함께 구매했습니다" (효율 중심)
- **고향마켓**: "서울에서 신안이 그리우신가요? 어머니가 담근 것 같은 새우젓입니다" (감성 중심)
- 사용자의 고향, 계절, 명절 등을 반영한 맞춤 추천

#### 3) 판매자-구매자 직접 소통
- 쿠팡은 판매자와 구매자의 직접 소통을 차단
- **고향마켓은 전화·채팅으로 직접 소통 가능**
- "고향 소식 전하면서 물건도 사는" 관계형 쇼핑

#### 4) 협동조합식 운영 (수익 재분배)
- 수수료 5% 이하 (쿠팡 대비 1/3 수준)
- 수익의 일정 부분을 판매자에게 환원
- 군민이 주인인 플랫폼

### 3-3. "고향마켓" 브랜드 확장성

```
신안군 고향마켓  ←  1단계: 파일럿
무안군 고향마켓  ←  2단계: 인근 확대
목포시 고향마켓
   ...
전국 228개 시군구  ←  3단계: 전국 확대
```

- **하나의 앱, 하나의 브랜드**로 전국 228개 시군구에서 공통 사용
- 앱 안에서 "신안군" 선택 → 신안군 고향마켓
- **신안군이 전국 최초 시작 지역**이라는 상징성

## 4. 신안군 특화 전략

### 4-1. "1004섬" 브랜드 연계

| 전략 | 내용 |
|------|------|
| 관광 + 쇼핑 연계 | 퍼플섬 방문 관광객 → 앱으로 재구매 유도 |
| 섬별 특산물 스토리 | "증도 천일염", "비금도 시금치", "흑산도 홍어" 등 섬 이름이 곧 브랜드 |
| 계절별 기획전 | 봄(대파), 여름(천일염), 가을(김), 겨울(새우젓) |
| 선물 세트 | 명절 고향 선물 세트 (택배 직송) |

### 4-2. 귀농·귀촌 청년 연계

- 앱 관리·마케팅을 귀농 청년 일자리로 연결
- 어르신 디지털 교육 → 청년 마을 활동가 역할
- 청년-어르신 세대 연결의 플랫폼

### 4-3. K-푸드 수출 연계 (장기)

- 해외 교민 시장 (뉴욕·LA·도쿄 한인)
- "신안 천일염" = K-솔트 프리미엄 브랜드화
- 전복·김 등 수출 경쟁력 있는 품목 중심
